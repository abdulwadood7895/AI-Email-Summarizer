AI Email Summarizer

Overview

AI Email Summarizer is an automation workflow built using n8n and Google Gemini AI.

The workflow:

1. Monitors new unread emails in Gmail.
2. Sends email content to Gemini AI.
3. Generates a concise summary and action items.
4. Sends the summarized result back to Gmail.

Tech Stack

- n8n
- Google Gemini API
- Gmail API

Workflow

Gmail Trigger → Gemini AI Agent → Gmail Send Message

Features

- Automatic email summarization
- Action item extraction
- Gmail integration
- AI-powered workflow automation
